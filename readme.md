# install npm
` npm init -y `

## install express
` npm i express `

## install nodemon
` npm i -D nodemon `

### setup nodemon in package.json file

```js
nodemon app.js
```
## install mongoose js
`npm i mongoose`

## install dotenv
`npm i dotenv`
## install jsonwebtoken
`npm i jsonwebtoken`
## install bcrypt
`npm i bcrypt`
## install nodemailer
`npm i nodemailer`
## install cors
`npm i cors`
